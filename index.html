PowerDNS on Rails by Kenneth Kalmer

Ruby on Rails based management software for PowerDNS's generic MySQL and PostgreSQL backends.

Dependencies

  • Ruby & Rubygems
  • MySQL or PostgreSQL
  • PowerDNS

Install

  1. git clone git://github.com/kennethkalmer/powerdns-on-rails.git
  2. cd powerdns-on-rails
  3. vi config/database.yml
  4. rake db:seed
  5. ruby script/server
  6. Point your browser to http://localhost:3000/
  7. Login with 'admin' and 'secret'
