PowerDNS on Rails by kennethkalmer

Ruby on Rails based management software for PowerDNS's generic MySQL and PostgreSQL backends

Dependencies

Ruby & Rubygems MySQL or PostgreSQL PowerDNS (for testing)

Install

git clone git://github.com/kennethkalmer/powerdns-on-rails.git cd powerdns-on-rails vi config/database.yml rake db:seed ruby script/server
